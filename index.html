Welcome to jogajog Joomla Contact Form.

jogajog is a Joomla Extension for Contacting from the joomla administrator panel.

How to Install JOGAJOG

Download Latest Zip File From here http://adidac.github.com/jogajog/ . Login to your joomla administrator panel go to Extension Manger from Extensions Menu then click on Select or Browse File Icon select your recently downloaded JOGAJOG Zip File. Then Click on Install.After successfull installation Click on Extension Manager Menu after that select Module manager you can see Joomla Jogajog Manager Menu click here and start jogajog(means - Communication between people).

jogajog Features?

  • You can customize the contact form what field to be diaplayed on form.
  • You can enable or disable captcha .
  • you can choose any captcha like recaptcha,coolphpcapctcha,textcaptcha
  • You can customize the the color of the form.
  • You can customize the any text on the form according to your need.
  • You can block any browser from displaying the form.
  • You can block any IP from using the form.
  • You can block any HTML Tags.
